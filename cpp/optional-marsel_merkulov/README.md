# Optional

В этом задании вам необходимо реализовать `Optional`.

Интерфейс вы найдёте в файле `src/optional.h`.
Поведение должно соответствовать [`std::optional`](https://en.cppreference.com/w/cpp/utility/optional/optional).

Обратите внимание, что:

- `Optional` должен стремиться повторять наличия и тривиальности special member'ов (формальные требования описаны на cppreference);
- Все операции `Optional` должны поддерживать использование в constexpr контексте.

При выполнении этого задания нельзя пользоваться *constraints* (ограничениями через `requires` и эквивалентным синтаксисом с концептами).
